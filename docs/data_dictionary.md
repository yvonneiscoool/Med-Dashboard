# Data Dictionary — App-Layer CSV Files

These files live in `data/app/` and are the sole data interface for the Tableau dashboard. They are generated by `src/marts/export.py`.

---

## app_overview.csv

**Grain:** review_panel + year (one row per panel per year)
**Source mart:** `mart_panel_year.parquet`
**Max rows:** ~5,000

| Column | Type | Description |
|--------|------|-------------|
| review_panel | string | FDA review panel code (e.g., "AN", "CH", "SU") |
| year | int | Calendar year (2019-2025) |
| event_count_raw | int | Total adverse event device records (all versions) |
| event_count_dedup | int | Deduplicated adverse event count (latest version only) |
| death_related_reports | int | Events with at least one patient death |
| serious_injury_reports | int | Events with serious injury outcomes |
| malfunction_reports | int | Events with device malfunction |
| recall_count | int | High-confidence-mapped recall count |
| class_i_recall_count | int | Class I (most severe) recall count |
| class_ii_recall_count | int | Class II recall count |
| class_iii_recall_count | int | Class III recall count |
| clearance_count | int | 510(k) clearances in that panel + year |
| events_per_100_clearances | float/null | Normalized event rate; NULL if clearances < 10 |
| recalls_per_100_clearances | float/null | Normalized recall rate; NULL if clearances < 10 |
| severe_recall_share | float/null | Class I / total recalls; NULL if total < 3 |

## app_category_product.csv

**Grain:** product_code + year (one row per product code per year)
**Source mart:** `mart_product_code_year.parquet` enriched with `dim_product_code`
**Max rows:** 20,000 (top N by event volume if exceeded)

| Column | Type | Description |
|--------|------|-------------|
| product_code | string | FDA product code (e.g., "DXY", "FRN") |
| year | int | Calendar year (2019-2025) |
| event_count_dedup | int | Deduplicated adverse event count |
| recall_count | int | High-confidence-mapped recall count |
| class_i_recall_count | int | Class I recall count |
| clearance_count | int | 510(k) clearances |
| events_per_100_clearances | float/null | Normalized event rate |
| recalls_per_100_clearances | float/null | Normalized recall rate |
| recall_to_event_ratio | float/null | Recalls per event; NULL if events = 0 |
| severe_recall_share | float/null | Class I / total recalls; NULL if total < 3 |
| device_name | string | Device name from classification table |
| review_panel | string | FDA review panel code |
| device_class | string | Device class (1, 2, or 3) |
| medical_specialty_description | string | Human-readable medical specialty name |

## app_manufacturer.csv

**Grain:** manufacturer + product_code + year
**Source mart:** `mart_firm_product_year.parquet` enriched with `dim_product_code`
**Max rows:** 30,000 (top N by event volume if exceeded)

| Column | Type | Description |
|--------|------|-------------|
| manufacturer | string | Manufacturer name as reported in source data (partially standardized via pipeline) |
| product_code | string | FDA product code |
| year | int | Calendar year (2019-2025) |
| event_count_dedup | int | Deduplicated adverse event count for this firm + product + year |
| recall_count | int | High-confidence-mapped recall count |
| severe_recall_count | int | Class I recall count |
| firm_share_within_product | float/null | Firm's share of events within product_code + year |
| firm_share_within_panel | float/null | Firm's share of events within panel + year |
| device_name | string | Device name from classification table |
| review_panel | string | FDA review panel code |

## app_methodology.csv

**Grain:** source + metric (key-value pairs)
**Purpose:** Data coverage and quality metadata for the Methodology dashboard page
**Typical rows:** ~10-15

| Column | Type | Description |
|--------|------|-------------|
| source | string | Data source identifier (e.g., "adverse_events", "recalls", "mapping_coverage") |
| metric | string | Metric name (e.g., "row_count", "date_min", "date_max", "event_product_code_coverage_pct") |
| value | string/numeric | Metric value |

### Expected rows:

| source | metric | Description |
|--------|--------|-------------|
| adverse_events | row_count | Total cleaned adverse event records |
| adverse_events | date_min | Earliest event date |
| adverse_events | date_max | Latest event date |
| recalls | row_count | Total cleaned recall records |
| recalls | date_min | Earliest recall date |
| recalls | date_max | Latest recall date |
| clearances | row_count | Total cleaned 510(k) records |
| clearances | date_min | Earliest clearance date |
| clearances | date_max | Latest clearance date |
| dim_product_code | row_count | Total product codes in dimension table |
| mapping_coverage | event_product_code_coverage_pct | % of event product codes that match classification |
